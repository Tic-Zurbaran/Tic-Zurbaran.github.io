Funciones para las imagenes

El módulo image contiene funciones para leer y grabar imágenes, como así también para transferir superficies a formatos accesibles para otros paquetes.

Note que no hay una clase Image, una imagen se carga como un objeto Surface. La clase Surface permite operaciones como dibujar lineas, pintar pixeles, capturar regiones, etc.

El módulo image es una dependencia importante de pygame, aunque el soporte extendido a formatos es opcional. Por defecto solo puede cargar imágenes BMP sin compresión; pero con pygame se provee un soporte de imágenes completo. Se pueden cargar los siguiente formatos: JPG, PNG, GIF (sin animación), BMP, PCX, TGA, TIF, LBM, PBM, XPM.

  • Cargar Imagen: Para poder cargar la imagen usaremos: imagen = pygame.load("ruta_imagen")
  • Mostrar Imagen: Se muestra la imagen: pantalla.blit(imagen(posx, posy))
